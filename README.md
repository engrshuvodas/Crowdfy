> **Crowdfy** is a pixel-perfect recreation of a modern, dark-themed SaaS landing page. It showcases a premium design aesthetic with glowing gradients, glassmorphism effects, and responsive layout techniques.

## üåü Features

-   **Dark Mode Esthetic**: Deep space background with vibrant purple accents.
-   **Glassmorphism Cards**: Modern frosted glass effects for feature and pricing cards.
-   **Interactive Elements**:
    -   Sticky header with backdrop blur on scroll.
    -   Smooth fade-in animations for content.
    -   Hover effects on buttons and cards.
-   **Responsive Design**: Fully responsive layout that adapts seamlessly to mobile, tablet, and desktop screens.
-   **Custom Graphics via CSS**: creative use of CSS to simulate complex UI elements (dashboards, charts) without external images.

## üõ†Ô∏è Tech Stack

-   **HTML5**: Semantic structure and accessibility.
-   **CSS3**:
    -   Flexbox & Grid layouts.
    -   CSS Variables for theming.
    -   Advanced gradients and box-shadows.
    -   Media queries for responsiveness.
-   **Vanilla JavaScript (ES6+)**:
    -   Intersection Observer API for scroll animations.
    -   DOM manipulation for sticky headers.

## üöÄ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

You need a modern web browser (Chrome, Firefox, Safari, Edge) to view the project.

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/crowdfy-landing-page.git
    ```
2.  **Navigate to the project directory**
    ```bash
    cd crowdfy-landing-page
    ```
3.  **Open `index.html`**
    -   Simply drag and drop `index.html` into your browser.
    -   OR use a live server extension (vs-code) to run it locally.
